Detailed Description

Sensorless Rotor Sector Sensor.

Note
This sensor CANNOT be used with field oriented control as it uses back EMF to determine the rotor sector rather than by using a voltage/current observer. It is intended to provide a simplistic way of using sensorless control with trapezoidal control.
